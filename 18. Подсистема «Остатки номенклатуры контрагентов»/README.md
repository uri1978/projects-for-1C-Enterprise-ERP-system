Система предназначена для хранения остатков товара у поставщика, с последубющим выводом в формах номенклатуры, подбора номенклатуры, в карточке товара. Есть возможность автоматической загрузки остатков товара из предоставленных xls файлов на почтовый ящик. Программа согласно расписания будет забирать такие письма и по правилам загружать остатки в базу.
